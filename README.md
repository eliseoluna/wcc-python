# Python Projects

This repository contains a collection of small Python homework projects for CIS 202. Each script focuses on a different beginner-to-intermediate programming concept such as user input, conditionals, loops, functions, file handling, and classes.

## Requirements

- Python 3

## Main Projects

### `caeser-cipher`
Encrypts or decrypts a message using a Caesar-style letter shift of 2 characters.

### `file-stats-analyzer`
Reads numeric data from a file and calculates:

- Number of values
- Minimum
- Maximum
- Average
- Variance
- Standard deviation

When running the program, enter the file path exactly as needed from your current folder, for example:

```bash
folder/example.txt
```

### `atm-simulator`
Implements a simple ATM-style banking program using a `BankAccount` class. The program allows the user to:

- Check balance
- Deposit money
- Withdraw money
- Quit the program


## Notes

- Most projects are interactive and require user input in the terminal.
- These scripts are written as individual homework assignments, so they are meant to be run separately rather than as one combined application.
